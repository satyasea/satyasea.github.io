  • Beaufort

    • The Beaufort sea weather app translates hard to understand raw data from multiple buoys into meaningful sea condition observations and warnings.
    App was previously published on Android Market by Androids of Mars as Mars Sea Conditions GPS.
    Code reuse: reuses some modified updated GPS connection code from web tutorials, and some modified example code from a beginner Android book from around 2011.

  • SpeedFactor

    • The SpeedFactor app is a speedometer for yacht racing with audio feedback, so captain and crew do not have to look at their speedometer, and can hear immediate results to their sailing inputs.
    Free download page published on linkedin post 1 year ago first posted here with an update post.
    Code reuse: App Reuses some modified code from tutorials at Simplified Coding and reuses some modified updated GPS connection code. Published code reuse blog on Medium in How to make an App on November 10, 2020.
